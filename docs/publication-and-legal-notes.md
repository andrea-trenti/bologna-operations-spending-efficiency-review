# Publication and Legal Notes

This repository is configured for a **clean public GitHub setup**.

## What is published here

- original analytical Markdown papers;
- original Python scripts;
- source documentation and download guidance;
- no bundled municipal raw files.

## Why raw files are not bundled

The public GitHub version is intentionally source-linked rather than raw-data-bundled.
This approach has three advantages:

1. it keeps the repository lighter and easier to navigate;
2. it avoids unnecessary re-hosting of official municipal PDFs and datasets;
3. it preserves reproducibility through official links and explicit download instructions.

## Attribution logic

The Comune di Bologna legal notes indicate that the site content is released under **Creative Commons Attribution 4.0 International (CC BY 4.0)**, which requires attribution when material is shared or adapted.

Official reference page:
- `https://www.comune.bologna.it/informazioni/note-legali-v03`

For the open-data side, the repository therefore follows a prudent public-publication logic:
- cite the official source pages;
- keep the data download path explicit;
- avoid packaging the raw files directly in the public repository unless there is a deliberate reason to do so.

## Practical public GitHub recommendation

The current package is suitable for public GitHub publication as-is because it contains:
- original text;
- original code;
- source links;
- no bundled raw files.

## Important caveat

This is a practical publication note, not legal advice.
