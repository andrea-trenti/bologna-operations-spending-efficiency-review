# Publication Notes

This repository contains original analytical writing and original code built on official public sources.

## What is published here

- original Markdown papers;
- original Python scripts;
- source documentation and download guidance;
- no bundled municipal raw files.

## Why the raw files are not included

Keeping the repository focused on writing, code, and source documentation makes it easier to read and lighter to maintain. It also avoids unnecessary re-hosting of official municipal PDFs and datasets when the official pages are already public and clearly documented.

## Attribution logic

The project cites the official municipal pages and data portals used in the analysis. Source pages and expected local filenames are documented in `docs/data-sources.md`.
